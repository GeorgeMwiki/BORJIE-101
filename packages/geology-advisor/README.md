# @borjie/geology-advisor

Geology advisor. Reads drill-hole assays + collar/survey records and
vein samples from the LMBM and derives interpreted ore-body
primitives: composited drill intervals, vein triangulated meshes, and
contained-tonnage + grade statistics. Writes back ore-body model
facts (mesh, tonnage, grade-distribution) so the cost-engineer and
mine-planner advisors can consume them downstream. Pure geometry +
statistics; ports for LMBM and logger.
